Feature Status Note

Core Purpose

To automate daily tasks (app launching, media control, system management) through natural voice commands, while continuously learning from user corrections to improve accuracy over time.

Self-Learning

Adapts to user vocabulary by remembering corrections ("no, I mean...").

Hybrid Architecture

Low-latency local wake word detection (openwakeword) + High-accuracy cloud STT (Google).

Floating UI

A modern, non-intrusive "pill" interface that provides visual feedback (listening, processing, awake).

Data-Driven

Scans the local system to index apps, files, and folders for instant access without hardcoded paths.

Intelligent Core

Advanced natural language processing that understands context, intent, and corrections.
